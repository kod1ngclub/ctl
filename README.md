# CTL

C99 Template Library

## Concept

- STL-like library in C99
- Evaluated in compile time

## Getting started

Launch a wizard tool with `make.py`

```bash
git clone https://github.com/kod1ngclub/ctl.git

cd ctl
python3 make.py
```
